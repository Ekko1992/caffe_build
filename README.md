# Environment_build
shell script used for building Caffe and Sunergy runtime environment
====================================================================
dependency packages
-----------------------
Sunergy:
cuda 9.0
cudnn 7.0

Caffe:
cuda 9.0
cudnn 7.0
opencv 3.3

NOTE: caffe can be easily installed on Ubuntu 17.04 and later version by using:  
sudo apt install caffe-cpu  
sudo apt install caffe-cuda
